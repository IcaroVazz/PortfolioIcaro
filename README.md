🌌 Icaro Vaz | Personal Portfolio

Portfólio pessoal desenvolvido para apresentar minha trajetória como Desenvolvedor Full Stack, reunindo projetos, experiências e competências técnicas em uma experiência visual moderna, interativa e imersiva.

O projeto foi construído com foco em performance, identidade visual e experiência do usuário, combinando uma arquitetura front-end limpa com animações fluidas e elementos gráficos dinâmicos inspirados no conceito Aurora Emerald.

Destaques do Projeto

- Interactive Galaxy Engine
 Background desenvolvido com HTML5 Canvas, contendo partículas e estrelas interativas que respondem dinamicamente ao movimento do cursor, criando profundidade e sensação de imersão visual.

- Aurora Emerald Design System
 Identidade visual baseada em tons escuros de Slate combinados com detalhes em Teal e Emerald, proporcionando uma estética moderna e tecnológica.

- Glassmorphism UI
Componentes translúcidos utilizando backdrop-blur para integração harmoniosa entre interface e ambiente visual.

- Experiência Responsiva
Layout otimizado para diferentes dispositivos, garantindo navegação fluida tanto em desktop quanto em mobile.

- Arquitetura Moderna
Estrutura componentizada e escalável voltada para organização, manutenção e alta performance.

Tecnologias Utilizadas
- React.js (Vite)
- Tailwind CSS
- Lucide React
- HTML5 Canvas API
- CSS Animations & Keyframes
  
 Objetivo

O portfólio foi desenvolvido para consolidar identidade profissional, apresentar projetos de forma interativa e demonstrar conhecimentos em desenvolvimento front-end moderno, UI/UX e experiências web imersivas.
